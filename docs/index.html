Features

Everything you need,
nothing you don't.

No accounts. No tracking. No bloat. Just fast, reliable word lookup and page summarization that works on every website.

Double-Click Definitions

Double-click any word or select a phrase. Get structured definitions with phonetics, audio pronunciation, examples, synonyms, and antonyms.

Smart Page Summaries

Press Ctrl+K or right-click to summarize any page. Works offline with TF-IDF extraction, or connect an AI provider for smarter results.

Context-Aware

Captures the surrounding paragraph so definitions match how the word is actually used, not just the generic dictionary entry. AI mode makes this even smarter.

Zero Dependencies

No npm packages. No build step. No supply chain risk. Everything uses native browser APIs: Web Crypto, Shadow DOM, Intl.Segmenter, Fetch.

Encrypted Storage

API keys are encrypted at rest with AES-GCM using a PBKDF2-derived key (600,000 iterations). Per-device salt. Nothing ever leaves your browser.

YouTube Transcripts

Extracts and summarizes YouTube video transcripts. Multiple extraction strategies handle format changes. Supports auto-generated captions.

How it works

Three interactions, infinite knowledge.

1

Double-click

Click any word or select a phrase. A tooltip appears instantly with definitions, phonetics, and synonyms.

2

Ctrl+K

Open the command palette to summarize the current page or look up any word by typing it.

3

Right-Click

Select any text and choose "Summarize selection" from the context menu.

+

AI (Optional)

Connect Gemini, OpenAI, or any custom endpoint for context-aware definitions and smarter summaries.

Definition Sources

Five APIs, one smart fallback chain.

Each source is tried in order. Multi-word phrases use a different order optimized for concepts and proper nouns.

SourceWhat it providesType
Dictionary API Phonetics, audio, definitions, examples, synonyms, antonyms Rich Free
Datamuse Definitions + parallel synonym/antonym fetch Rich Free
Wiktionary Encyclopedic definitions, rare and multi-language words Free
Wikipedia 3-sentence summary for concepts and proper nouns Free
Wikipedia Search Full-text search fallback for unknown phrases Free
